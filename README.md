# Nuvemshop no FECBR 2026: preview da landing page

Prévia navegável da landing page institucional do Dia 1 da Arena D2C no Fórum E-Commerce Brasil 2026.

## Escopo desta versão

- HTML, CSS e JavaScript vanilla.
- Experiência responsiva para desktop e mobile.
- Formulário demonstrativo aberto em modal; nenhum dado é enviado.
- Assets visuais usados exclusivamente para avaliação desta proposta.

## Executar localmente

Abra `index.html` ou sirva a pasta com um servidor HTTP estático.
