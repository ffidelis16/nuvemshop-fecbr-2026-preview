# Nuvemshop no FECBR 2026: preview da landing page

Preview público sanitizado da landing page institucional para a ação fechada da Nuvemshop no Dia 1 do Fórum E-Commerce Brasil 2026.

## Escopo desta versão

- HTML, CSS e JavaScript vanilla.
- Experiência responsiva para desktop e mobile.
- Formulário demonstrativo aberto em modal; nenhum dado é enviado.
- Assets visuais usados exclusivamente para avaliação desta proposta.

Versão atual: V4.5, revisada a partir do retorno da cliente em 15/07/2026. A V4.4 enviada para a primeira revisão está preservada na tag `v4.4-client-review`.

## Executar localmente

Abra `index.html` ou sirva a pasta com um servidor HTTP estático.
